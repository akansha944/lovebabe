# Love Babe & Bean

A standalone marketing website for [Love Babe & Bean](https://www.instagram.com/lovebabeandbean/) — a cosy hole-in-the-wall coffee and matcha shop in Newmarket, Auckland.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001).

## Deploy to Vercel

This folder is a self-contained Next.js app. To deploy separately:

1. Create a new GitHub repository
2. Copy this `love-babe-and-bean` folder into it (or push only this folder)
3. Import the repo in [Vercel](https://vercel.com) and deploy

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript
